# Hi! 👋

## Die Idee
Wäre doch ganz cool wenn wir üben könnten hilfreiches Feedback zu geben und fremden code zu lesen (und 🤓 zu verdienen!)

## Workflow
### eigenen code reviewn lassen
- erstellt von hier einen neuen Branch
- pusht dort euren Code
- sagt bescheid :)


### fremden code reviewn
- branch von Interesse pullen
- Kommentare als ``` # comment ``` einfügen (am besten in neue Zeilen, nicht daneben)
- pushen
- Bescheid geben
